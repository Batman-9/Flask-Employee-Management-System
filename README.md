# Employee Management REST API

A simple Employee Management REST API built using **Python**, **Flask**, and **MySQL**. This project demonstrates CRUD (Create, Read, Update, Delete) operations using REST APIs and stores employee information in a MySQL database.

---

## 🚀 Features

- View all employees
- Add a new employee
- Update employee details
- Delete an employee
- JSON-based API responses
- MySQL database integration
- RESTful API design
- Error handling for invalid requests

---

## 🛠️ Technologies Used

- Python 3
- Flask
- MySQL
- mysql-connector-python
- Postman (API Testing)
